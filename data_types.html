Data Types

Primitive Data Types

Primitive types store single values and are immutable (cannot be changed directly).

  • String - Represents text enclosed in quotes
  • Number - Represents numeric values (integers or decimals)
  • BigInt - Represents large integers beyond Number limit
  • Boolean - Represents logical values
  • Undefined - A variable declared but not assigned
  • Null - Represents an empty or unknown value
  • Symbol - Used to create unique identifiers

Non-Primitive (Reference) Data Types

These types store collections or complex structures and are mutable.

  • Object : Key-value pairs
  • Array : Ordered list of values
  • Function : Block of reusable code
  • Date : Represents dates and times

Checking Data Types

Use the typeof operator

note :-
