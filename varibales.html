Variables

variable is a container used to store data that can be used and changed throughout a program. It acts as a named memory location that holds a value such as a number, string, object, or boolean.

Declaring Variables

  1. var - Function scoped
  2. let - Block scoped
  3. const- Block scoped

Variables can store different data types.

Variable Scope

  • Global scope: Accessible everywhere.
  • Function scope: Accessible only inside a function (var).
  • Block scope: Accessible only within {} block (let, const).
