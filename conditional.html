conditionals

Conditionals in JavaScript are statements that help your program make decisions based on certain conditions. They check whether an expression is true or false and execute different blocks of code accordingly.

1. if Statement

Used to execute a block of code only if a condition is true.

2. if...else Statement

Runs one block if the condition is true, another if itâ€™s false.

3. if...else if...else Statement

Used when you have multiple conditions to check.

4. switch Statement

Used as an alternative to multiple if...else if statements. It compares a single expression against multiple possible values.

5. Ternary Operator (?:)

A shorthand way to write an if...else statement. It returns a value based on a condition.
