Functions

A function in JavaScript is a block of reusable code designed to perform a specific task. Instead of writing the same code repeatedly, you can define a function once and call it whenever needed.

Syntax:

function functionName(parameters) {
// code to execute
return value; // optional
}

Function Declaration

This is the traditional way to define a function.

Hoisted: You can call it before or after its declaration.

Function Expression

You can also assign a function to a variable.

Not hoisted — you must define it before using it.

Arrow Function (ES6)

A shorter syntax introduced in modern JavaScript.

Function Parameters and Arguments

Parameters: Variables listed in the function definition.
Arguments: Actual values passed to the function.

Return Statement

Functions can return values to the code that called them.

Default Parameters

You can set default values for parameters.

Nested Functions

Functions can be defined inside other functions.

Anonymous Functions

Functions without a name — often used as callbacks.

Spread Operator (...)

Used to expand (or “spread out”) elements of an array or object. It helps in copying, combining, and passing elements easily.

Rest Operator (...)

Used to collect multiple arguments into a single array or object. It works in the function parameter list or object/array destructuring.

Note:

...spread → expands elements (used in arrays/objects/functions).
...rest → gathers elements (used in parameters/destructuring).
