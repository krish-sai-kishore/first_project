Loops

Loops in JavaScript are used to execute a block of code repeatedly until a specified condition is false. They make your program more efficient by avoiding repetition.

1. for Loop

Used when you know how many times you want to run a block of code.

for (initialization; condition; increment/decrement)
{
// code to be executed
}

2. while Loop

Used when you donâ€™t know the number of iterations in advance. It keeps running as long as the condition is true.

while (condition) {
// code block
}

3.do...while Loop

Executes the code at least once, even if the condition is false, then continues looping while the condition remains true.

do {
// code block
} while (condition);

4. for...in Loop

Used to loop through the properties of an object.

for (let key in object) {
// code block
}

5.for...of Loop

Used to iterate over arrays, strings, or any iterable object.

for (let value of iterable) {
// code block
}

Note

  • for in return numbers

  • for of return values in a arrays,string,etc
